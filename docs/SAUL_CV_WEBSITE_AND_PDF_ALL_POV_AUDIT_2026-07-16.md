# Saul CV Website and PDF - All-Perspective Critique and Revision

## Website

- Human reader: the previous overhaul repeated a focus dropdown, eight large cards, more-role links, download controls and tuning controls. The revised card uses one compact colour-coded focus rail, one live experience composition and one primary action.
- Visual design: the site keeps its mild spectrum, portrait, translucent surface and geometry. Colour remains semantic on the website rather than becoming a decorative résumé treatment.
- Recruiter: role, profile, selected experience and skills appear before download mechanics.
- Applicant: every focused URL opens the same card with the correct view already selected.
- Accessibility: native links, buttons, details and checkboxes preserve keyboard and screen-reader behaviour.
- Mobile: the focus rail becomes a two-column compact grid and the preview becomes one column.
- SEO: focused routes retain unique metadata, canonical URLs and ProfilePage/Person structured data.
- Maintenance: all routes use one CSS file, one JavaScript file and one canonical data file.
- Preservation: the map and complete archive remain in the ordinary main-page flow; duplicate archive download controls are removed.

## PDFs

- Hiring manager: the former rainbow/geometric PDF confused the website identity with an application document. The revised PDF uses one consistent professional monochrome template.
- Visual hierarchy: name, role, contact, profile, experience, skills, credentials and languages have distinct levels without coloured boxes or decorative geometry.
- Readability: the one-page role outputs use restrained density, a clear date rail and consistent typography.
- ATS: all designed PDFs contain extractable text, and every focus retains a separate single-column ATS version and plain-text version.
- Factuality: Montreal remains limited to the earlier Pizza Pizza crew role; the later Assistant Manager role remains Toronto; BUMI is corrected; Farsi is advanced; historical credentials remain labelled historical; latte art stays absent.
- Academic scope: MA and education appear only in the Education output or an Education-selected web state.
- Consistency: all 12 role focuses use the same professional template rather than role-specific colour styling.
