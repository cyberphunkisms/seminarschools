Editorial Log

Schema versions, routing decisions, bar-and-verification decisions, refusals.

The catalog accumulates not only entries but decisions about entries. This file is the public record of those decisions: what was added, what was rejected, what was rerouted, what was left undone because completing it would have required fabrication.

Each entry carries a date-anchor, a status badge (shipped / blocked / routing-decision), and a brief account. Refusals are first-class: an honest catalog documents what it did not catalog and why.

Bar & verification decisions

SHIPPED · U1-U8

The 8 untranslatable entries (saudade, sehnsucht, mamihlapinatapai, komorebi, schadenfreude, sobremesa, fika, tsundoku) carry verification badges: ◉ primary read (translator/author engaged source directly), ◐ secondary verified (passes through reputable scholarly intermediary), ◌ cited from bibliography (entered via citation chain, not direct read). Frazer/Smith pattern for contested types (May 2026)
SHIPPED · H6 ml*

Methodology entry H6 codifies the cite-pair-the-contested rule: when an archetypal type is itself disputed (e.g., the "scapegoat king" inherited from Frazer's Golden Bough but criticized by Jonathan Z. Smith), the catalog cites BOTH the founding author and the contesting author rather than picking a side. The reader gets the live contest, not a settled adjudication.

Worked example in catalog: scapegoat-king tag carries Frazer 1890-1915 alongside J. Z. Smith 1973 contesting reference. Reader can follow either lineage.

Rule #1 CITES + LOGOI (May 2026)
SHIPPED · core* slot 1

Top-priority core memory rule: every factual claim requires a verified primary source. Just-knowing without source-check is Q5 (objection delivered without source-check when the source-check would kill it). SEO content does not count as scholarship. Wikipedia is starter-only, never cite. LOGOI clause: ml* / mc* / bb* / core* outputs must be universal-truth-bearing; soft-pleasing is a violation.

Catalog enforcement: every entry added since this rule's installation cites primary sources by publisher / ISBN / journal-volume. The aa* refinement track this session (A12 / A21 / C5) all met this bar; META_ENTRIES cite Lévi-Strauss / Wittgenstein / Foucault primary works by Gallimard / Blackwell publication, SETTINGS_LIBRARY cites Eliade / Bachelard / Foucault / Vernant primary works.

On verification, the Schmidt referenced is Victoria Lynn Schmidt (45 Master Characters, Writer's Digest 2001, rev. 2012, ISBN 978-1599635347), already in the catalog as taxonomy schmidt45. Her categories explicitly use Bolen's light/shadow Greek-pantheon framework: "Aphrodite (Seductive Muse / Femme Fatale)" / "Zeus (King / Dictator)" / "Hera (Matriarch / Scorned Woman)", these are Bolen 1984/1989 archetypal pairs extended into a writer's toolkit. Adjacent gap noted (not yet fixed): the Bolen 1989 work (Gods in Everyman) does NOT have its own taxonomy entry in the catalog, only the 1984 Goddesses in Everywoman.
