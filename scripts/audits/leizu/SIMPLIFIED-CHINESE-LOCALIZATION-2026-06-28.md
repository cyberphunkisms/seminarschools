# Simplified Chinese localization

The `简` control now presents the current revised Leizu home page in Simplified Chinese.

This pass localizes the Simple English status bar and controls, page controls, navigation, hero, teaching approach, course catalogue, route controls, sessions, pricing, teacher section, FAQs, access section, contact section, all 24 course descriptions, and Five Elements glosses.

The page uses `lang="zh-Hans"` while Simplified Chinese is active. The French and Traditional Chinese localizations remain unchanged. No booking, payment, intake, policy, or server-side pipeline code changed.
