# Persian localization

The **فا** control now renders the current Leizu home page in Persian.

This update includes the revised public page, navigation and accessibility controls, the Simple English banner, the course picker, all current course descriptions, Five Elements glosses, plans, teacher section, parent FAQs, access information, and the Chinese consultation route.

The page uses `lang="fa"` and the existing right-to-left layout rules. Payment, intake, Stripe, Cal.com, and Resend code remain unchanged.
