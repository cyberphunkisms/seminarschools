# French localisation

The FR control now translates the Leizu home page, including the control strip, navigation, hero, course catalogue, course descriptions, plans, payment shapes, FAQs, access information, contact block, and booking modal.

The English source remains the fallback for the later Traditional Chinese, Simplified Chinese, and Farsi passes.

The Stripe, Netlify, Cal.com, and Resend code paths were left unchanged.
