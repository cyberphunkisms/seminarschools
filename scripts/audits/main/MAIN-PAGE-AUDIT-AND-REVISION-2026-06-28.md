# Main Page Audit and Revision

## Role

`/main/` now functions as the conceptual centre of Seminar Schools. The root homepage remains the interactive map and practical first entry. The central constellation jewel links here.

## Changes applied

- Added three first-screen routes for Leizu Academy, The Agora, and Teacher Resources.
- Tightened the threshold, lecture, academy, founder, community, writing, contact, and portfolio copy without changing the stated portfolio terms, rate, or enrollment structure.
- Reduced the top navigation to the routes that help a first-time visitor act.
- Limited the main-page lecture teaser to six upcoming events. The full archive remains at `/polymythseminars/`.
- Replaced automatic footer looping and top scroll resistance with an explicit return link.
- Reduced the background geometry from three seventh-depth recursions to three sixth-depth recursions with a larger minimum radius.
- Quieted ornamental flowers in the default viewing mode.
- Added a visible geometry control. Pointer movement produces a small geometry parallax without a continuous animation loop. Reduced-motion preferences keep the field still.
- Restored the homepage central jewel destination to `/main/`.

## Remaining editorial decisions

- Decide whether the portfolio application on this page remains the preferred Leizu entry or whether `/leizu/` should become the only tutoring intake route. Both are currently active and serve different visitor preferences.
- Decide whether Creative Mode should remain as an optional visual register. It remains available because it belongs to the existing page identity, while the default page now prioritizes readability.
- The current page carries English content only. A later localisation pass can follow the same route as the Leizu language work if `/main/` becomes a primary public landing page for multilingual families.
